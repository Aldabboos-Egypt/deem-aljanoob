برنامج العماله المنزلية

وصف البرنامج

يعمل برنامج الخدم مع برنامج المحاسبة بشكل متكامل وذلك عند تسجيل كل عمليه علي برنامج الخدم يقوم بارسال العملية المحاسبيه المناسبه لها الي برنامج المحاسبه وهذا بالاضافه الي اذا قام الموظف بالغاء العمليه لاداعي لالغاء القيود المحاسبيه التي تم انشائها عند العمليه الاساسيه لان برنامج الخدم سوف يقوم بارسال القيود العكسيه في هذه الحاله ومن هنا يعمل برنامج الخدم وبرنامج المحاسبه معا بشكل متكامل ولاداعي حينئذ لمحاسب كفؤ للتسجيل المحاسبي

يمكنك من خلال برنامج المحاسبه طباعه التقاريير المحاسبيه المرتبطه بالنظام كما يمكنك اغلاق الدوره المحاسبيه شهريا لسهوله تنفيذها

واخيرا عندما يكون هناك برنامج خدم مرتبط ببرنامج المحاسبه يمكنك اداره اي نشاط اخر مع نشاط الخدم وتسجيلهم علي شجره حسابات واحده للنظام
